## Use Sensor
#### 介绍
IIC拓展、UART可控拓展控制，读取传感器数据。
>IIC拓展芯片：TCA9548A
>IO拓展芯片：PCF8574
>通断控制器：CD4066

#### 硬件控制
01_Hardware:
>01_XIAOBLE-CORE：拓展在Seeed XIAO BLE 上的使用。
>02_IIC-UART：独立模块，可以根据自己的喜好进行IIC/UART拓展。
